11.27 - Second Team Meeting

Key issues were identified and discussed during the second meeting:

  1. Classification of Context Meaning:

    • Original meaning, reflects physical condition (Label: 0).
    • Original meaning, does not reflect physical condition (Label: 1).
    • Sarcasm or figurative meaning (Label: 2).

  2. Keyword Filtering:

    • Should certain keywords be removed?

  3. Missing Attached Images:

    • How to address the issue of missing attached images in Weibo posts?

  4. Dataset Balance:

    • Examine the dataset to determine if it is balanced. If imbalanced, increase data for underrepresented labels.
